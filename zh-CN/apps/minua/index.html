通过学习获得屏幕时间。

功能介绍

赚取屏幕时间

孩子通过使用家长选定的学习应用，获得娱乐应用的访问权限。可配置比例从 1:1 到 12:1。

Apple Screen Time API

使用 Apple 的内置框架，在操作系统层面实现可靠的应用限制。

允许使用时段

家长可设置每日时间窗口（例如 15:00–20:00），在此期间娱乐应用可被解锁。超出时间段则保持锁定。

每日重置

时间余额每天午夜自动重置。孩子每天早晨都能从零开始。

PIN 保护设置

所有家长设置均由 4 位 PIN 码保护，多次输入错误后将锁定。孩子无法更改规则。

无需账户

无需电子邮件、注册或云同步。设置好应用后将设备交给孩子即可。一切数据保留在设备上。

隐私与数据

  • 所有使用数据通过 iOS 加密存储在本地。
  • 无账户、无云端、无服务器。
  • Screen Time API 通过 Apple 的框架完全在设备上运行。
  • 没有可被传唤、可被入侵或可被出售的数据。

适合哪些用户

Minua 适合正在寻找儿童屏幕时间应用、无追踪家长控制应用，或希望将学习时间与娱乐屏幕时间有效结合的家长。对于不喜欢监控型家长控制软件的家庭尤为合适。

为什么选择 Minua？

赚取式屏幕时间模式

比一般家长控制更具针对性：Minua 将屏幕时间重新定义为孩子通过有意义的活动赢得的奖励。

Apple 原生控制

使用 Apple 自有的 Screen Time 框架，比许多跨平台监控应用提供更简洁、更值得信赖的体验。

默认保护隐私

无账户、无云端，深受注重隐私的家长青睐——他们不希望家庭习惯或孩子数据存储在第三方服务器上。

更适合家庭使用的理念

奖励学习比单纯围绕屏幕限制进行的惩罚循环更容易解释，也更易于日常实践。

常见问题

Minua 会监视孩子吗？

不会。本应用有意基于设备端控制，而非依赖追踪密集型监控软件。

孩子使用学习应用能赚取时间吗？

是的。这一"赚取时间"机制是产品的核心概念。

需要额外创建家庭账户吗？

不需要。Minua 基于本地设置和 Apple 系统框架运行，无需管理额外的云端账户。
