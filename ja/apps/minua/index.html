Minua

学習でスクリーンタイムを獲得できます。

App Storeでダウンロード
デバイス上で完結すべてのデータはローカルに保存
Apple Screen Time内蔵のペアレンタルコントロール
アカウント不要登録なしで利用可能

主な機能

スクリーンタイムを獲得

子どもは保護者が選んだ学習アプリを使った時間に応じて、娯楽アプリへのアクセスを獲得できます。比率は1:1から12:1まで自由に設定可能です。

Apple Screen Time API

AppleのOS標準フレームワークを使用し、信頼性の高いアプリ制限を実現します。

許可する時間帯

保護者は娯楽アプリを解放できる1日の時間枠（例：15時〜20時）を設定できます。その時間外は娯楽アプリはロックされたままになります。

毎日リセット

時間のバンクは毎日深夜にリセットされます。子どもは毎朝ゼロから新たにスタートできます。

PIN保護の設定

保護者のすべての設定は4桁のPINで保護されており、入力ミスが続くとロックアウトされます。子どもはルールを変更できません。

アカウント不要

メールアドレスも登録もクラウド同期も不要です。アプリを設定して子どもにデバイスを渡すだけ。すべてのデータはデバイス上に保存されます。

プライバシーとデータ

  • すべての利用データはiOSの暗号化によりローカルに保存されます。
  • アカウントなし、クラウドなし、サーバーなし。
  • Screen Time APIはAppleのフレームワークを通じてデバイス上で完全に動作します。
  • 差し押さえられるものも、漏洩するものも、売られるものも何もありません。

対象ユーザー

Minuaは、子ども向けスクリーンタイムアプリや追跡なしのペアレンタルコントロールアプリを探している保護者、または学習時間とレクリエーション向けのスクリーンタイムをうまく結びつけたいと考えている方のためのアプリです。監視色の強いペアレンタルコントロールソフトに抵抗を感じるご家庭に特に適しています。

Minuaを選ぶ理由

獲得型スクリーンタイムモデル

一般的なペアレンタルコントロールより具体的な価値提案です。Minuaはスクリーンタイムを「子どもが有意義な活動を通して獲得するもの」として捉え直します。

Apple純正の制御機能

AppleのScreen Timeフレームワークを使用することで、多くのクロスプラットフォーム監視アプリよりも信頼性の高い体験を提供します。

デフォルトでプライベート

アカウントもクラウドも不要なため、家族の習慣や子どものデータを外部サーバーに置きたくないプライバシー重視の保護者に最適です。

家族に優しいアプローチ

学習を報酬として与えるアプローチは、スクリーン制限だけを軸にした罰則の繰り返しよりも、説明しやすく日常に馴染みやすいです。

よくある質問

Minuaは子どもを監視しますか？

いいえ。このアプリはトラッキングを多用した監視ソフトではなく、デバイス上のコントロールを中心に設計されています。

学習アプリを使うことで時間を獲得できますか？

はい。この「獲得型タイム」の仕組みが製品の核となるコンセプトです。

別のファミリーアカウントが必要ですか？

いいえ。Minuaはローカル設定とAppleのシステムフレームワークを基盤としており、新たなクラウドアカウントの管理は不要です。
