Cognithix vs Atom Learning: お子さまに合う11+アプリはどちら?

2026年3月 · 読了10分 · 教育

11+の準備を進めているなら、Atom Learning はほぼ確実に目にしたことがあるでしょう。英国全土で数万の家庭に使われている、この分野で最も知名度の高いサービスの一つです。同じ問題に全く異なるアプローチで取り組む Cognithix を見かけたこともあるかもしれません。どちらも11+試験を深く理解した開発者が作った本格的な対策ツールです。ただし、設計思想が異なり、カバーする範囲も異なり、向いている家庭も異なります。これは両者を正直に比較したものです。それぞれの弱点についても触れています。

開示事項: 私たちは Cognithix を開発しています。Atom Learning の強みについてできるだけ公平に書くよう努めましたが、決める前にどちらも試してみることをお勧めします。選択肢の全体像を把握したい場合は、アプリ、参考書、家庭教師、それらの組み合わせ方を網羅した 11+対策完全ガイド もご覧ください。

各アプリの概要

Atom Learning

Atom Learning はブラウザで動作するウェブベースのプラットフォームです。2018年からサービスを提供しており、教師が作成したコンテンツを大量に蓄積しています。主な機能は以下のとおりです。

  • 科目: 英語、数学、言語推理 (VR)、非言語推理 (NVR)
  • 問題数: 経験豊富な教師が作成した7万問以上
  • 動画レッスン: 練習前に概念を説明する短い解説動画
  • アダプティブ学習: 成績に応じて問題の難易度が変化
  • マルチデバイス対応: 最新ブラウザがあれば、ノートPC、タブレット、スマートフォンのいずれでも利用可能
  • 保護者向けレポート: 進捗レポートと成績サマリー
  • 料金: サブスクリプション制。プランにより月額約£48〜56

完成度が高く、実績のあるプラットフォームです。特に動画レッスンは多くの家庭から支持されており、概念を理解してから練習問題に取り組みたいお子さまに有効です。

Cognithix

Cognithix は、根本的に異なるアプローチをとるネイティブ iOS アプリです。固定の問題バンクから出題するのではなく、アルゴリズムエンジンがデバイス上で問題を生成します。主な機能は以下のとおりです。

  • 科目: 英語、言語推理 (VR)、非言語推理 (NVR)、言語適性テスト (LAT)
  • 問題形式: 4科目合計で42種類の異なる出題形式
  • 問題生成: デバイス上のアルゴリズムが毎回新しい問題を生成するため、同じ問題が繰り返されることはありません
  • アダプティブ難易度: 科目ごとに独立して調整されるため、英語が得意でも NVR が苦手なお子さまは両方で適切なレベルの問題に取り組めます
  • 保護者ダッシュボード: 正答率、スピード、つまずきのパターンを追跡。早押しの当てずっぽうや正答率の低下も検出します
  • プラットフォーム: iOS (iPhone および iPad)
  • オフラインアクセス: インターネット接続なしで全機能を利用可能
  • プライバシー: すべてのデータはデバイス上に保存。外部サーバーへの送信はありません
  • 料金: 買い切り制

動画レッスンはなく、Apple デバイスのみ対応です。ただし、他のどの11+アプリも現時点でカバーしていない領域を扱っています。

LAT という決め手

これが両アプリの最大の差別化要因です。家庭によっては、この一点で選択が決まるかもしれません。

お子さまの志望校の入学試験に 言語適性テスト (LAT) が含まれている場合、対策の選択肢は大幅に絞られます。LAT は見慣れない言語を即座に習得する能力を測るテストで、架空の語彙、構築された文法体系、パターンベースの言語課題が出題されます。毎年意図的に内容が変わるよう設計されており、11+の中でも特に対策が難しい部分の一つです。

Atom Learning は現時点で LAT を扱っていません。これは批判ではなく、ほとんどの11+プラットフォームも同様です。LAT は専門の語学校が採用しているニッチな試験要素であり、テスト自体が「見慣れない素材」を前提としているため、対策コンテンツを作ること自体が本質的に難しいのです。

Cognithix は LAT を扱っています。実際のラテン語語彙と文法に基づく演習、習得可能なルールを持つ構築言語、隠れた単語の検出、架空言語の数体系、パターンベースの言語タスクが含まれています。問題は固定のバンクではなくアルゴリズムで生成されるため、毎回新しい LAT 素材で練習できます。これは本番試験の体験と一致しています。本番でも、すべての素材が設計上「見慣れないもの」だからです。

お子さまの試験に LAT が含まれている場合、現時点で対応しているのは Cognithix のみです。LAT が含まれていない場合、この優位性は当てはまらず、比較はより複雑になります。

Atom Learning が優れている点

Atom Learning が本当に優れている点を正直に伝えなければ、読者の役に立ちません。以下の点では Atom Learning が上回っています。

動画レッスン。 これは Atom Learning の最大の特徴であり、Cognithix に同等のものはありません。お子さまが初めて触れる概念、たとえば見たことのない種類の言語推理問題などに対して、Atom の短い動画レッスンは練習前にアプローチを説明してくれます。最初の説明が必要なお子さまには、これは実質的な価値があります。

マルチデバイス対応。 Atom はブラウザで動作するため、Windows ノートPC、Chromebook、Android タブレット、iPad、スマートフォンで使えます。Cognithix は Apple デバイスが必要です。ご家庭が Android 環境だったり、お子さまが学校の Chromebook で練習している場合、Atom の方が現実的な選択です。

数学のカバー範囲。 Atom Learning は英語、VR、NVR に加えて数学のカリキュラムも含んでいます。Cognithix は数学を扱っていません。数学も含めて一つのアプリで完結させたい場合、Atom の方が総合的なパッケージです (ただし LAT は未対応です)。

大きなコミュニティと実績。 Atom Learning は2018年から運営されており、非常に多くの家庭に使われています。保護者のレビュー、フォーラムの議論、実際の成果データが豊富に存在します。その歴史が信頼感と社会的証明を生んでおり、新しいアプリが短期間で積み上げられるものではありません。

教師が作成したコンテンツ。 Atom の問題バンクの全問題は、経験豊富な教師が執筆しています。アルゴリズム生成の問題より手書きコンテンツの質を好む保護者もいます。特に英語の読解問題では、素材文の質が重要です。

考慮すべき点: 月額£48のサブスクリプションを18か月間継続すると£864になります。買い切り制はその名のとおりです。正しい選択は、準備期間の長さと動画解説の価値をどう評価するかによって異なります。

Cognithix が優れている点

Cognithix に本質的な優位性がある点を以下に示します。

LAT 対応。 上で述べましたが、改めて強調します。お子さまが言語適性テストの対策を必要としている場合、Cognithix はそれを提供する唯一のアプリです。LAT 演習には、ラテン語由来の語彙学習、構築言語の文法、隠れた単語の検出、架空の数体系、パターンベースの言語推理が含まれています。現時点で、他のアプリはこれを提供していません。

オフラインアクセス。 Cognithix はインターネット接続なしで動作します。問題はデバイス自体で生成されるため、電車の中、待合室、旅先、あるいは Wi-Fi が使えない場所でも練習できます。Atom Learning はすべてのセッションでインターネット接続が必要です。

無制限の新問題。 問題がアルゴリズムで生成されるため、問題が尽きることはありません。同じ問題に繰り返し触れて答えを覚えてしまうリスクがありません。毎回のセッションが本当に新しい素材です。これは、数か月にわたって大量に練習するお子さまにとって特に重要です。固定の問題バンクは、どれほど大きくても最終的には使い切ってしまいます。

買い切り制の料金。 一度支払うだけです。継続的なサブスクリプションも、解約の手間も、18か月の準備期間中に積み上がるコストもありません。予算を重視する家庭や、サブスクリプションモデルを好まない家庭にとって、これは実質的な違いです。

プライバシー。 Cognithix の処理はすべてお子さまのデバイス上で完結します。練習データ、成績履歴、個人情報が外部サーバーに送信されることはありません。子どものデータプライバシーへの関心が高まる中、デバイス上での処理は本質的に保護性の高いアプローチです。Atom Learning は、ほとんどのウェブプラットフォームと同様にデータをクラウドに保存します。

より深い保護者インサイト。 Cognithix の保護者ダッシュボードは単純な進捗追跡を超えています。気づきにくいパターンを検出します。早押しの当てずっぽう (問題を読んでいない可能性があります)、セッション中の正答率低下 (疲労や集中力の低下のサイン)、特定の問題形式における継続的な弱点などです。これらのパターンにより、数週間の非生産的な練習の後に問題を発見するのではなく、適切なタイミングで介入できます。

どちらを選ぶべきか

一つの正解はありません。最善の選択は、お子さまの具体的なニーズ、志望校の試験形式、家庭の実情によって異なります。以下は判断の参考となるフレームワークです。

Atom Learning を選ぶ場合...

  • 新しい概念を動画で説明してもらってから練習した方が理解が深まるお子さまの場合
  • マルチデバイス対応が必要な場合 (特に Windows、Chromebook、Android)
  • 志望校の試験に LAT が含まれていない場合
  • 英語や推理と同じアプリで数学もカバーしたい場合
  • 長い実績を持つ確立されたプラットフォームの安心感を重視する場合

Cognithix を選ぶ場合...

  • 志望校が言語適性テストを課している場合
  • 移動中、旅行中、あるいは信頼できるインターネット環境がない場所でお子さまに練習させたい場合
  • 継続的なサブスクリプションより買い切り制を好む場合
  • 子どものデータプライバシーを重視し、デバイス上での処理を望む場合
  • お子さまが数か月練習を続け、他のプラットフォームで新しい素材が不足してきた場合
  • 基本的な進捗グラフを超えた保護者インサイト、急ぎ・疲労・特定の弱点の検出を求める場合

両方を検討する場合...

  • 予算に余裕があり、LAT を含む幅広い科目を試験でカバーする必要がある場合
  • 動画で学ぶ数学と英語 (Atom) と、無制限の LAT および推理練習 (Cognithix) の両方を求める場合
  • お子さまが非常に多く練習し、長い準備期間で固定の問題バンクを使い切る可能性がある場合

この2つのアプリは実際に補い合う関係にあります。Atom Learning の動画レッスンと数学コンテンツは、Cognithix がカバーしないギャップを埋めます。Cognithix の LAT 対策、オフライン機能、無制限の問題生成は、Atom がカバーしないギャップを埋めます。両方を使うことは重複ではなく、それぞれのツールを最も得意な領域で活用することです。

まとめ

11+対策市場は、ここ数年で大きく改善されました。Atom Learning も Cognithix も、お子さまの時間と保護者の投資を尊重するよく作られたツールです。どちらも魔法の解決策ではなく、継続的な練習、十分な休息、穏やかな家庭環境がアプリの選択よりもはるかに重要です。

まだ迷っている場合は、両方の無料トライアルを試してみてください。お子さまがそれぞれのアプローチにどう反応するか観察してください。動画解説で伸びるお子さまもいれば、すぐに練習に入りたいお子さまもいます。オフラインアクセスが必要なお子さまもいれば、常に Wi-Fi 環境にいるお子さまもいます。最善のアプリは、これからの数か月間の準備期間を通じて、お子さまが自分から進んで継続的に使うものです。

11+の準備を進めていますか?

Cognithix は英語、VR、NVR、言語適性テストの4科目すべてをカバーしています。
