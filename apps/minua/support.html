Frequently Asked Questions

How does it work?

Parents set up Minua on their child's device. They choose which apps count as "learning" and which count as "fun". Fun apps are locked by default. When the child uses learning apps, they earn screen time for fun apps based on the ratio the parent set (e.g. 6 minutes learning = 1 minute fun).

How do I set up Minua?

Open Minua and follow the onboarding steps: select learning apps, select fun apps, set the earn-to-spend ratio, configure allowed hours, and create a 4-digit parent PIN. Minua will request Screen Time permission during setup.

Why does Minua need Screen Time access?

Minua uses Apple's Screen Time API to monitor which apps are being used and to show shields on locked apps. This is the same system Apple uses for its own Screen Time parental controls. All processing happens on your device.

Can my child bypass the shields?

The shields use Apple's built-in Screen Time framework, which is the same technology behind Apple's own parental controls. For extra protection, we recommend disabling app deletion in Settings and setting Auto-Lock to 1 minute to prevent idle earning.

What is the parent PIN for?

The PIN prevents your child from changing the app selections, ratio, allowed hours, or resetting the app. It is stored as a hash on the device, not as plaintext.

I forgot my parent PIN

Uninstall and reinstall Minua to reset all settings, including the PIN. This will clear all time bank data.

When does the time bank reset?

The earned and spent time resets automatically at midnight each day.

Where is my data stored?

All data is stored on your device only. No accounts, no cloud, no servers. Deleting the app permanently deletes all data.

What devices does Minua support?

Minua requires iPhone or iPad running iOS 17 or later. Screen Time features require a physical device (they do not work in the iOS Simulator).
