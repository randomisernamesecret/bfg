Durch Lernen verdienen

Minua

Bildschirmzeit, die Kinder sich durch Lernen verdienen. Kinder schalten Unterhaltungs-Apps frei, indem sie Zeit in den Lern-Apps verbringen, die Sie auswählen — im Verhältnis, das Sie festlegen. Alles bleibt auf Ihrem Gerät.

So funktioniert Minua

Einmal die Regeln festlegen, dann das Gerät übergeben. Kinder sehen genau, wie viel Zeit sie verdient haben und welches Lernen mehr freischaltet — keine Überwachung, kein Nörgeln, nichts wird irgendwohin gesendet.

Was die App kann

Ein vollständiges Verdienen-statt-Verbieten-Werkzeug für Bildschirmzeit — Kinder verdienen sich den Zugang zu Unterhaltungs-Apps durch Lernen, und Eltern behalten die Kontrolle, ohne ein weiteres Konto verwalten zu müssen.

Bildschirmzeit verdienen

Kinder erhalten Zugang zu Unterhaltungs-Apps, indem sie Zeit in den von den Eltern ausgewählten Lern-Apps verbringen — im einstellbaren Verhältnis.

Auf Apples Kindersicherung aufgebaut

App-Sperren werden auf Systemebene durchgesetzt — zuverlässige Einschränkungen auf Betriebssystem-Niveau, ohne separaten Überwachungsdienst.

Erlaubte Zeiten

Legen Sie ein tägliches Zeitfenster fest, in dem Unterhaltungs-Apps freigeschaltet werden können. Außerhalb dieser Zeiten bleiben sie gesperrt.

Täglicher Reset

Das Guthaben an verdienter Zeit wird jeden Tag zurückgesetzt, sodass Kinder jeden Morgen mit einem leeren Konto frisch starten.

PIN-geschützte Einstellungen

Die Elterneinstellungen sind durch eine private PIN mit Sperre nach Fehlversuchen geschützt. Kinder können die Regeln nicht ändern.

Kein Konto erforderlich

Keine E-Mail, keine Registrierung, keine Cloud-Synchronisation. Einrichten, Gerät übergeben — alles bleibt auf dem Gerät.

Privat von Haus aus

Die meisten Kindersicherungs-Apps verlangen ein Familienkonto und leiten die Routinen Ihres Kindes durch eine Anbieter-Cloud. Minua nicht. Es ist auf Apples eigener Kindersicherung aufgebaut und läuft vollständig auf Ihrem Gerät.

  • Alle Nutzungsdaten bleiben lokal auf dem Gerät.
  • Keine Konten, keine Cloud, keine Server, kein Tracking.
  • App-Sperren werden auf Systemebene auf dem Gerät durchgesetzt.
  • Einmalkauf — kein Abo.

Für wen es gedacht ist

Minua ist für Eltern, die eine Bildschirmzeit-App für Kinder, eine Kindersicherung ohne Tracking oder einen besseren Weg suchen, Lernzeit mit Freizeitnutzung zu verbinden. Besonders relevant für Familien, denen überwachungslastige Kindersicherungssoftware unangenehm ist.

Verdientes Bildschirmzeit-Modell

Konkreter als generische Kindersicherung: Minua macht Bildschirmzeit zu etwas, das Kinder durch sinnvolle Aktivität verdienen.

Apple-native Steuerung

Der Aufbau auf Apples eigener Kindersicherung erzählt eine sauberere, vertrauenswürdigere Geschichte als viele plattformübergreifende Überwachungs-Apps.

Standardmäßig privat

Kein Konto und keine Cloud — ideal für datenschutzbewusste Eltern, die Kinderdaten nicht auf einem fremden Server speichern möchten.

Besserer Ansatz für die Familie

Lernen zu belohnen lässt sich leichter erklären und im Alltag umsetzen als endlose Strafen rund um Bildschirmzeitlimits.

Häufige Fragen

Überwacht Minua Kinder?

Nein. Minua setzt auf Steuerung direkt auf dem Gerät, nicht auf überwachungslastige Tracking-Software.

Verdienen Kinder Zeit durch Lern-Apps?

Ja. Dieses Verdienst-Prinzip ist das Kernkonzept des Produkts.

Ist ein weiteres Familienkonto nötig?

Nein. Minua basiert auf einer lokalen Einrichtung, nicht auf einem weiteren Cloud-Konto, das verwaltet werden muss.
